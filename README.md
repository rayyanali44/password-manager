# Password Manager

A full-stack Password Manager built using the MERN stack. It allows users to securely store, view, update, copy, and delete passwords through a clean and responsive interface.

## Features

- Add new passwords
- View saved passwords
- Update existing passwords
- Delete passwords with confirmation
- Copy passwords to clipboard
- Password visibility toggle
- Toast notifications
- Data stored in MongoDB
- Responsive UI built with React and Tailwind CSS

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rayyanali44/password-manager.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a `.env` file in the backend directory with your MongoDB connection string.

4. Start the frontend and backend servers.
